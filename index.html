Languages and Tools
C++
The foundation of systems programming. Total memory control, blazing fast. Painful to learn but nothing else gives you this kind of control.
Python
Goes from idea to working code faster than anything. Data, automation, ML -- it does it all without getting in your way.
C#
Great for APIs, game dev with Unity, anything .NET. Statically typed and actually enjoyable to write.
JavaScript
The language of the web. You can love it or hate it. You're still writing it. Runs everywhere, breaks creatively.
TypeScript
JavaScript but it tells you when you broke something before you find out at 2am. Makes large codebases survivable.
SQL
Decades old, still essential. Know this well and you'll never be blocked by your data layer.
React
The most used way to build interactive UIs. Component-based, fast, ecosystem covers basically everything.
Node.js
JavaScript on the server. One language, full stack. Fast for I/O-heavy work and async by default.
Docker
"Works on my machine" -- solved. Packages your app and everything it needs so it runs the same way everywhere.
